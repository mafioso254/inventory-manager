# Inventory Manager

## Flask app (existing)

1. Set up virtual env and install requirements:
   - `python -m venv venv`
   - `venv\Scripts\Activate.ps1` (PowerShell)
   - `pip install -r requirements.txt`
2. Run:
   - `python run.py`
3. Visit:
   - `http://localhost:8080` (main)
   - `http://localhost:8080/qb-clone` (QuickBooks style clone)

## Next.js + Tailwind app

1. `cd nextjs`
2. `npm install`
3. `npm run dev`
4. Visit:
   - `http://localhost:3000`
   - `http://localhost:3000/qb-clone`
